1. Go to the path where the java file is placed in the command prompt.
2. First compile the code using command javac Id_main.java
3. Keep the data set files in the same folder where the java file is placed 
4. Then you can execute the code using java Id3main followed by the arguements eg: java Id_main 4 training_set.csv validation_set.csv test_set.csv 1
5. Here in the above command, 4 represents no of nodes to prune and 1 represents whether to print the tree.
